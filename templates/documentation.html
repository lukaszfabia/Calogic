Documentation of Calculator
Expression Accepted expression Alternative notation
factorial x! write a number and then click button with factorial
square root sqrt(x) insert number after this use sqrt button next to input
cube root cbrt(x) insert a number in the same way as above
floor floor(x) integer part of a number, inserting is like other with brackets
ceil ceil(x) round-up of a number, inserting is like other with brackets
absolute value |x| absolute value of a number, inserting is like other with brackets
inversion 1/x inversion of a number, there is no shortcut version
natural logarithm ln(x) natural logarithm of a number, insert number then click button
logarithm with base log(x,y) logarithm of a number where x is a base of log, insert first base then number with comma like in pattern
n-root nrt(x,y) root of y, degree x, there is also the same method as above
sinus sin(x) sinus of a number, write argument of function and then click
cosinus cos(x) cosinus of a number, write argument of function and then click
tangens tan(x) tangens of a number, write argument of function and then click
cotangens cot(x) cotangens of a number, write argument of function and then click
secans sec(x) secans of a number, write argument of function and then click
cosecans csc(x) cosecans of a number, write argument of function and then click
arcsinus arcsin(x) arcsinus of a number, write argument of function and then click, note that it should be a number from [-1 1]
arccosinus arccos(x) arccosinus of a number, write argument of function and then click, note that it should be a number from [-1 1]
arctangens arctan(x) arctangens of a number, write argument of function and then click
arccotangens arccot(x) arccotangens of a number, write argument of function and then click
exponential e^x takes exponent of x, insert power and then expression
power x^y power of x, degree y, there is no method to insert in alternative option
